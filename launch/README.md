# Launch Assets

This folder is the go-to-market package for HackYourAgent.

Use it to:

- position the repo against adjacent tools like promptfoo, garak, inspect_ai, and agentdojo
- record a short proof-driven demo
- publish a Show HN post
- publish an X thread and LinkedIn post
- run a disciplined launch checklist instead of improvising

## Files

- `positioning.md`
- `demo-script.md`
- `show-hn.md`
- `x-thread.md`
- `linkedin-post.md`
- `launch-checklist.md`
